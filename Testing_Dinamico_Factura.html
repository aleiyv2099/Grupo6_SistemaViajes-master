Observaciones / Recomendaciones

1. Registro de Factura: El sistema ejecuta correctamente la transacción completa (INSERT factura + UPDATE reservas + INSERT detalle). Se recomienda validar en la interfaz que el cliente tenga reservas activas antes de habilitar el botón de emisión.
2. Cálculo del Monto: El cálculo automático funciona correctamente al seleccionar el cliente. Se recomienda mostrar el detalle de reservas incluidas junto al monto para mayor trazabilidad.
3. Búsqueda / Listado: El filtro responde correctamente ante búsquedas con y sin resultados. Se recomienda agregar un mensaje explícito "No se encontraron registros" cuando la tabla quede vacía.
4. Eliminación Lógica: La eliminación lógica preserva la integridad referencial de la base de datos. Se recomienda solicitar confirmación al usuario antes de ejecutar la operación para evitar eliminaciones accidentales.
5. Generación de PDF (RF-6): Se recomienda validar que el directorio src/main/java/pdf/ exista antes de intentar escribir el archivo, o capturar el error con mensaje descriptivo al usuario.
